# rgaa-scraper
Récupère les règles RGAA à partir du site officiel et les enregistre au format JSON.
Le fichier JSON généré est disponible dans le dossier dist : [RGAA3.json](https://github.com/SylvainBzh/rgaa-scraper/blob/master/dist/RGAA3.json)

Si vous souhaitez créer à nouveau ce fichier, suivez la procédure d'installation.

# installation
Cloner le dépot. Puis dans le dossier d'installation lancer la commande :

`npm install`


Pour lancer l'éxécution:

`node .`

Un fichier RGAA3.json a été créé dans le dossier dist.
Si vous êtes derrière un proxy décommenter la ligne correspondante dans le fichier lib/index.js
